# hh_db
Это приложение предоставляет инструмент для поиска открытых вакансий в заданных компаниях,
создания базы данных PostgreSQL и предоставления методов отбора вакансий.

## Функциональность программы
Пакет src, содержащий следующие модули:

1. *companies_fetcher* - содержит класс **HeadHunterCompanies**. 
Поиск по ***id*** данного списка работодателей от HeadHunter.
2. *vacancies_fetcher* - содержт класс **Вакансии HeadHunter**.
Поиск ***открытых вакансий*** из указанного списка работодателей, предоставленного HeadHunter.
3. *db_manager* -  содержит класс **DBManager**.
Предоставляет методы для создания базы данных и таблиц PostgreSQL.
Заполнение таблиц информацией, предоставляемой классами **HeadHunterCompanies** и **HeadHunterVacancies**.
Предоставление методов, используемых в основной функции для работы с базой данных.
4. *logger.py* - настройка логгера.

### Как работает программа

+ Приложение запускается с запроса списка компаний, которые пользователь хотел бы найти. 
**Если оставить поле пустым** - будет выбран список компаний по умолчанию. (см. примечания)

+ Пользователь подключается к базе данных и создает таблицы для заполнения.

+ Поиск инициализируется. Обработанная информация заполняется таблицами

+ Пользователь выбирает один из вариантов в меню выбора.

#### Примечание

+ Список компаний по умолчанию следующий: ***(Pooling, Amigoweb, Doubletapp, InlyIT, PUSK, Digital Sail, Skillline, Mindbox, SPRINTHOST, Voximplant).***

+ Проект с использованием виртуальной среды ***Poetry***. Информация о зависимостях
находится в ***pyproject.toml***. 

Для плавной установки зависимостей рекомендуется использовать виртуальную среду ***Poetry***. 

Для установки зависимостей используйте команду:

```
poetry install
```
